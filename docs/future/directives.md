# Directives

:::info
This functionality is not yet implemented
:::

### Import source files

```mew
#load "foo.mew"
#load "stuff/*.mew"
```

### Reference Mew projects

```mew
#reference "../../foo.mewx"
#reference "../../bar.mewx"
```

### Executable name

```mew
#name "MyName"
```

:::note
Can only be specified in a `mewx` project file.
:::
